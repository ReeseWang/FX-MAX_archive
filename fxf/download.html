Download the trial version of FX Freeze 1.02 (657Kb)
This is a full featured version of FX Freeze with only one limitation - after a 14-day trial period you'll not be able to make new freezes. All songs and projects will load successfully.

System requirements:

  • Windows 98/ME/2000/XP
  • VST 2.0 compatible host (Cubase, Logic or SONAR, Project 5 (VSTis via VST adapter), etc.)
  • VST instruments and/or FX
    Hardware requirements depend on versions of Windows and host applications, but not lower than 300 MHz CPU, 64 MB RAM, 2 MB HD + HD space for frozen audio data.

    History:

    1.02
    New Features:

  • Remote Control. Trigger freeze by a MIDI Control Change events.
    Fixes:
  • Crackles with ultra-frozen plug-ins
  • Can't freeze several instances of the same plug-in
  • Updated manual for Track Freeze, Bounce, Offset features

    1.01
    New features:

  • Track Freeze (run FX Freeze w/o inner plug-in)
  • Bounce mode (plus 'bounce' button to use on hosts w/o full VST support)
  • Mute/MuteAll/Solo
  • Disk streamer options (buffer size, priority)
    Fixes:
  • Silence on Windows 98
  • Looping crackle
  • UAD-1 offset report for PDC
  • Quantum-fx resize
